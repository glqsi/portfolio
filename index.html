Power BI Embedded

Power BI Embedded

O Power BI Embedded permite integrar relatórios e dashboards em aplicativos personalizados. Com essa solução, é possivel criar uma experiência rica para os usuários, oferecendo a visualizações dos relatórios dentro da aplicação.

1. O que é Power BI Embedded e como ele pode beneficiar minha aplicação?

Power BI Embedded é uma solução da Microsoft que permite integrar relatórios e dashboards diretamente na aplicação web ou mobile. O portal foi desenvolvido utilizando frameworks e ferramentas de alta qualidade e open-source, o que permite uma grande redução de custos e mantendo a robustez, permitindo que os clientes acessem os relatórios de forma segura, sem necessidade de licenças adicionais do Power BI Pro. Isso proporciona uma visualização de dados personalizada, com permissões de acesso e com possíveis customizações.

2. Quais são os custos envolvidos para utilizar o portal?

O principal custo envolvido para utilizar o portal é o da máquina virtual onde o serviço está hospedado. Não há custos adicionais por usuário no portal, independentemente do número de acessos ou usuários. O custo é mais voltado para a infraestrutura necessária para suportar a aplicação, tornando a solução escalável e econômica.

3. Preciso de varias licenças Power BI Pro para os meus usuários do portal?

Não, você não precisa de varias licenças Power BI Pro para os usuários que acessam os relatórios no portal. Apenas uma licença Power BI Pro é necessária para publicar os relatórios e dashboards para o seu portal. Isso torna a solução mais acessível e flexível para negócios de diferentes tamanhos.

4. Como o Power BI Embedded é integrado ao meu portal web?

A integração do Power BI Embedded no portal web é feita por meio de APIs fornecidas pela Microsoft. Utilizando APIs, é possivel se comunicar, autenticar e incorporar os relatórios no portal, permitindo que os usuários naveguem com segurança. Toda a parte de autorização e permissões de acesso é feita com base em grupos. Com isso, é possível oferecer uma experiência personalizada e interativa para os usuários.
